# 🚀 Andy Chu | 朱廷翊 個人履歷專案 (2026 Edition)

這是一個為 **朱廷翊 (Andy Chu)** 量身打造的專業品牌網站，整合了資管系背景與量子計算、區塊鏈、人工智慧等多維度技術實力。

## 🛠 技術架構 (Tech Stack)
*   **Next.js 14 (App Router)**: 高性能 React 框架。
*   **TypeScript**: 型別安全開發環境。
*   **Vanilla CSS Modules**: 純粹且高效的賽博龐克風格樣式。
*   **Framer Motion**: 流暢的捲動與互動動畫。
*   **Lucide React**: 現代感極簡圖示庫。

## 📂 專案結構
*   `src/app/`: 路由與全域配置。
*   `src/components/`: 核心視覺組件 (Navbar, Hero, Skills, Projects)。
*   `src/components/*.module.css`: 每個組件對應的獨立樣式。

## 🏃 如何啟動開發環境
1.  開啟終端機 (PowerShell/CMD)。
2.  進入目錄：`cd andy-portfolio`
3.  執行指令：`npm run dev`
4.  在瀏覽器開啟：`http://localhost:3000`

## 🎨 視覺規範 (Theme)
*   **背景色**: `#050505` (深黑科技感)
*   **強調色**:

    *   紫色 (Purple): `#bc13fe` (區塊鏈、加密)
